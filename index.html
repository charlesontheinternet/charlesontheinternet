charlieinternets.eth is the web3 identity of Charlie Internets.

This website is hosted on the InterPlanetary File System (IPFS) - a peer-to-peer protocol designed to make the web upgradeable, resilient, and more open.

ETH.LIMO is used as a web 2.0 infrastructure stack for resolving Ethereum Name Service records and associated (IPFS) content, representing a bridge from the 'normal' internet to the decentralized one.
